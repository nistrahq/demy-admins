# Bibliography – Demy Android App

This file lists the main references and resources used during the development of the Demy Android mobile application.  
The focus is on Android development, Kotlin, Jetpack Compose, accessibility (a11y), and academic foundations.

---

## Official Documentation
- [Android Developers – Jetpack Compose](https://developer.android.com/jetpack/compose)
- [Material Design 3 Guidelines](https://m3.material.io/)
- [Kotlin Language Documentation](https://kotlinlang.org/docs/home.html)
- [Room Persistence Library](https://developer.android.com/training/data-storage/room)
- [Navigation in Compose](https://developer.android.com/jetpack/compose/navigation)
